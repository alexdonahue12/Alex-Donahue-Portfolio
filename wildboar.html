Case Study

Wild Boar: A One-Page Menu Design Project

This case study explores the design process behind "Wild Boar," a one-page menu created for a fictional BBQ restaurant. The project focused on establishing a strong typographic hierarchy and an engaging visual style that reflects the restaurant's bold and rustic identity.

Role
Typography, layout, print, grid
Course
Typography 2
Deliverables
One-page menu

Project brief

Project Brief: Design a one-page restaurant menu using Adobe In-Design. Choose a name, cuisine, and typographic style to reflect the restaurant's identity.

“Wild Boar” is a BBQ-themed menu with bold, fire-inspired typography and a rustic color scheme. I focused on creating a clear typographic hierarchy across seven levels to guide diners through the layout, using expressive type to convey the smoky, energetic character of the restaurant while keeping the menu readable and organized.
